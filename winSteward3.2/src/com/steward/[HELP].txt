윈집사 사용 설명서 ver.2.1.0

1. 프로그램 소개

 - 컴퓨터 어딘가에 저장된 파일이나 폴더를, 초기 저장 후 실행키 하나로 쉽게 실행할 수 있도록 돕는 프로그램 입니다.
 - 사이트도 실행키로 저장해둘 수 있습니다.
 - 포털사이트 검색의 경우, 검색어를 추가하여 실행하시면 검색이 된 페이지가 실행됩니다.
 - (ver.2.1.0)멜론 실시간 top 10 목록과, 이틀 전의 박스오피스 목록을 볼 수 있는 서브탭이 추가되었습니다.
 

2. 프로그램 시작

 2-1. 초기 설정 
 
	- 윈집사를 실행 하신 후, 우측 중단에 잇는 설정 버튼을 누르세요.

	2-1-1. 주소
		- 주소찾기 버튼은 해당 프로그램이 어느 폴더에 들어가있는지를 찾아볼 수 있는 버튼입니다.
		- 인터넷 사이트인 경우, 해당 url을 적어주세요. (작성 예 : www.naver.com)
		
	2-1-2. 실행키
		- 실행키 부분에 원하는 키워드를 작성하세요 (작성 예 : 네이버)
		
 	2-1-3. 단축키
 		- 단축키 지정을 원하시면 선택해주세요. 원하시지 않으실 경우 공백으로 저장하시면 됩니다.
 		- Ctrl, Alt, Shift + 키보드 숫자 or 영문자 한글자를 지원합니다. 원하는 키를 체크하시고(다중선택 가능), 키보드 부분에 숫자 or 영문자를 넣어주세요
 		(선택 예 : Ctrl + Shift + n)
 		
 	2-1-4. 카테고리
 		- 카테고리를  선택해주세요.(다중선택 불가)
 		- 나중에 내가 저장한 실행키를 잊으셨을 시, 카테고리 콤보박스에서 검색하시면 됩니다.

	- 주소, 실행키, 단축키, 카테고리 중 필요하신 부분을 모두 설정하셨다면 저장 버튼을 눌러주세요.
	- 설정 버튼을 잘못 누르셔서 들어오신 경우, 취소 버튼을 눌러주세요. 
 
 2-2. 실행
 	
	2-2-1. 메인탭
		- 텍스트 부분에, 저장하셨던 실행키를 작성 후 실행 버튼을 눌러주세요.
 		- 콤보박스는 지금까지 저장해두셨던 실행키들을 보여줍니다.
 		- 오른쪽 하단에는 순서대로
 			그림판, 커맨드, 메모장
 			캡쳐툴, 그림판, 탐색기
 			돋보기, 종료, 설명서 입니다.
		- 각 콤보박스에는 해당 카테고리에 맞는 실행키가 보여집니다.
	
	2-2-2. 서브탭(ver.2.1.0)
		- 첫번째 서브탭은 멜론 실시간 탑 10이 보여집니다.
		- 해당 곡을 누르면 그 곡에 대한 유튜브 검색 화면이 실행됩니다.
		- 두번째 서브탭은 한국영화진흥위원회의 이틀 전 박스오피스 정보가 보여집니다.
		- 해당 영화제목을 누르면 그 영화에 대한 네이버 검색 화면이 실행됩니다.
